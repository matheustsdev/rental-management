# Guia de Emissão de Recibos em PDF

Este guia explica as melhorias no processo de emissão de recibos para clientes e controle interno da loja.

## 📄 O que mudou no Recibo?

O novo recibo foi redesenhado para ser mais profissional, econômico e completo em termos de informações financeiras.

### 1. Novo Layout de Duas Vias (Economia de Papel)
Agora, ao gerar um recibo, o sistema criará automaticamente duas vias na mesma folha A4 (orientação paisagem):
- **Via do Cliente (Esquerda):** Para ser entregue ao cliente no momento do aluguel.
- **Via da Loja (Direita):** Para o controle interno e arquivo da loja.
*Dica: Você pode dobrar ou cortar a folha ao meio para separar as vias.*

### 2. Observações Diferenciadas (Privacidade e Clareza)
Para maior organização, as observações agora são separadas por via:
- **Na Via do Cliente:** Aparecerão **apenas** as "Observações para Recibo" (ex: regras de lavagem, prazos de devolução).
- **Na Via da Loja:** Aparecerão **apenas** as "Observações Internas" (ex: defeitos pré-existentes, anotações de costura).
*Isso garante que informações internas da loja não fiquem expostas ao cliente.*

### 3. Detalhamento Financeiro Completo
O recibo agora exibe claramente todos os valores envolvidos na transação:
- **Subtotal:** Soma dos valores de todos os produtos alugados.
- **Desconto:** Valor de desconto concedido ao cliente (se houver).
- **Sinal (Depósito):** Valor pago adiantado como garantia.
- **Valor Restante:** O saldo que o cliente ainda precisa pagar no momento da devolução ou retirada.

## 🚀 Como emitir o novo recibo

1. Acesse a página de **Aluguéis** no menu lateral.
2. Localize o aluguel desejado na lista ou utilize a barra de busca.
3. No card do aluguel, clique no botão **"Ver mais"**.
4. Selecione a opção **"Emitir recibo"**.
5. Aguarde um momento enquanto o sistema gera o documento (um ícone de carregamento será exibido).
6. O PDF será baixado automaticamente para o seu computador.

## ⚠️ Dicas e Cuidados
- **Produtos Longos:** Se o aluguel tiver muitos itens, o sistema criará automaticamente uma segunda página para garantir que nada seja cortado.
- **Limite de Texto:** O campo de observações tem um limite visual para garantir que os totais financeiros sempre apareçam na parte inferior da folha. Tente ser conciso para manter a estética do documento.
- **Erros de Geração:** Caso o sistema não consiga gerar o recibo, uma notificação de erro aparecerá no canto da tela. Verifique se todos os dados do aluguel (valores, datas e produtos) estão preenchidos corretamente.
